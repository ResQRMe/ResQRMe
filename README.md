# 🚑 RESQ ME

**RESQ ME** is an Emergency Medical Identification System designed to provide fast and secure access to critical health information during emergencies.

With the use of **QR Code and NFC technology**, users can store essential medical data such as:
- Allergies
- Medications
- Chronic conditions
- Emergency contacts

In case of an emergency, responders can simply **scan or tap** the QR/NFC tag to instantly access life-saving information.

---

## ⚡ Key Features

- 🚀 **Instant Access** – Retrieve medical data in seconds  
- 🔐 **Secure & Private** – Only essential information is stored  
- 📱 **Smart Technology** – QR Code + NFC integration  
- 🎯 **Easy to Use** – Simple setup and quick access  
- ❤️ **Life-Saving Purpose** – Built for real emergency situations  

---

## 💡 How It Works

1. Fill out your medical information  
2. Generate your QR/NFC profile  
3. Attach it to your keychain or card  
4. Scan or tap in emergencies  

---

## 🎯 Mission

To improve emergency response and help save lives by making critical medical information accessible anytime, anywhere.
